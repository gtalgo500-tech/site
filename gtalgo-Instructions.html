üì¶MT5 Installation & Setup

Before You Start

Make sure you have MetaTrader 5 installed and a trading account (demo or real).

Step 1: Download MetaTrader 5

Download MT5

Visit your broker's website or download directly from MetaQuotes:

https://www.metatrader5.com/en/download

Install and Open MT5

Run the installer and follow the on-screen instructions. After installation, open MetaTrader 5.

Create/Login to Trading Account

Use your broker credentials to log in, or create a demo account for testing.

Step 2: Install Your GTAlgo Bot

Download the EA File

After purchase, download the .ex5 file from your Whop dashboard.

Open MT5 Data Folder

In MT5, click: File ‚Üí Open Data Folder

This opens the MQL5 directory where all expert advisors are stored.

Copy the EA File

Navigate to: MQL5 ‚Üí Experts

Paste the downloaded .ex5 file into this folder.

Restart MT5

Close and reopen MetaTrader 5 to load the new Expert Advisor.

Verify Installation

Open the Navigator panel (Ctrl+N), expand "Expert Advisors", and confirm your bot appears in the list.

‚úÖ
Installation Complete!

Your GTAlgo bot is now installed and ready to use.

What is a Demo Bot?

Demo bots are free, limited versions that allow you to test the strategy in backtest mode before purchasing the full version.

How to Use Demo Bots

Download Demo Version

Visit the GTAlgo store and download the demo version of your chosen bot.

Install in MT5

Follow the same installation steps as above (File ‚Üí Open Data Folder ‚Üí MQL5 ‚Üí Experts).

Use in Strategy Tester

Demo versions can ONLY be used in the Strategy Tester (backtest mode). Press Ctrl+R to open the Strategy Tester and test the bot's performance on historical data.

‚ö†Ô∏è
Demo Limitations

Demo versions are restricted to backtest mode only. They cannot be used for live or demo account trading. Upgrade to the full or licensed version for complete functionality.

üîêLicense Activation (Licensed Version)

üîë
Licensed Bots Only

This section applies only to licensed versions with subscription-based licensing.

Step 1: Enable WebRequest in MT5

Open MT5 Options

Go to: Tools ‚Üí Options ‚Üí Expert Advisors

Add License URL

Check the box: "Allow WebRequest for listed URL"

Add this URL to the list:

https://ea-license-backend.gtalgo500.workers.dev

Save and Restart

Click OK and restart MetaTrader 5.

Step 2: Configure License in EA Settings

Drag EA to Chart

Drag your licensed bot from Navigator to the chart.

Enter License Credentials

In the EA settings window, find the "LICENSE SETTINGS" section and enter:

  • User Email: The email you used when purchasing
  • License Key: Your unique license key (received via email)

Enable AutoTrading

Make sure "Allow Algo Trading" is checked, then click OK.

Verify License

Check the "Experts" tab in MT5. You should see a success message confirming license validation.

‚úÖ
License Active!

Your bot is now licensed and ready to trade. The license will be validated automatically every 30 seconds.

‚ö†Ô∏è
Troubleshooting

If you see "License validation FAILED", make sure:

  • WebRequest URL is correctly added in MT5 settings
  • Email and License Key are entered exactly as received
  • Your license subscription is active
  • You have an active internet connection

Why Backtest?

Backtesting allows you to test the bot's performance on historical data before risking real money.

How to Run a Backtest

Open Strategy Tester

Press Ctrl+R or go to View ‚Üí Strategy Tester

Configure Test Settings

In the Strategy Tester panel:

  • Expert Advisor: Select your GTAlgo bot
  • Symbol: Choose the trading symbol (see recommended settings below)
  • Period: Select timeframe (M5, M15, H1, etc.)
  • Date Range: Choose testing period (recommended: 1+ year)
  • Model: Select "Every tick" for most accurate results

Adjust EA Parameters

Click "Expert properties" to customize settings like lot size, risk percentage, etc. (see bot-specific settings below).

Start the Test

Click the "Start" button and wait for the backtest to complete.

Analyze Results

Review the Results, Graph, and Report tabs to evaluate performance metrics like win rate, profit factor, drawdown, etc.

‚ö†Ô∏è
Important Notes
  • Always use high-quality historical data for accurate results
  • Past performance does not guarantee future results
  • Test on multiple timeframes and symbols for validation
  • Use realistic spread and commission settings
